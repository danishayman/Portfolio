## 📦 Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/danishayman/Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
